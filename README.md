## Install

```sh
curl -sL https://deb.nodesource.com/setup_0.12 | sudo bash -
sudo apt-get install git python-pip python-dev nodejs
sudo pip install tornado pymongo subprocess32
sudo useradd -m -s /bin/bash p2dv
sudo passwd p2dv
```
